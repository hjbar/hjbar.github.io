## Asteroids

<p style="text-align: justify;">
The goal of the game is to collect as many points as possible. To do this, the player must try to survive for as long as possible by destroying the asteroids heading their way.
</p>

<p style="text-align: justify;">
The game features a <a href="https://en.wikipedia.org/wiki/JavaScript">JavaScript</a> version and, if you aren't a fan of <a href="https://en.wikipedia.org/wiki/JavaScript">JavaScript</a>, another version is written in <a href="https://en.wikipedia.org/wiki/OCaml">OCaml</a> using <a href="https://en.wikipedia.org/wiki/Simple_DirectMedia_Layer">SDL</a>. This allows you to play either in your browser or locally.
</p>

<p style="text-align: justify;">
Both versions are written in <a href="https://en.wikipedia.org/wiki/OCaml">OCaml</a>. You can find the source code for this game on <a href="https://github.com/hjbar/jeu_asteroids">GitHub</a>.
</p>
